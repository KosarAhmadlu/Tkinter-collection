# Tkinter Learning Labs

A collection of practical desktop application projects built with **Python**, **Tkinter**, and **SQLite**.

This repository is designed to help beginners learn GUI programming by building real-world desktop applications instead of practicing isolated Tkinter widgets.

Each project focuses on a specific concept while introducing new GUI components, database operations, and application design principles.

---

## Projects


### 01. Library Management System

A desktop application for storing and managing book information.

**Concepts**

* Tkinter GUI
* SQLite Database
* CRUD Operations
* Search Functionality
* Listbox
* Database Management

---

### 02. Task Management System

A daily task management application with task status tracking.

**Concepts**

* Tkinter
* SQLite Database
* CRUD Operations
* Listbox
* Functions
* Task Status Management

---


### 03. Grade Management System

A desktop application for managing student grades.

**Concepts**

* Tkinter Basics
* Entry Widgets
* Buttons
* LabelFrame
* ttk.Treeview
* SQLite Database
* CRUD Operations
* Conditional Statements

---

### 04. Contact Management System

A simple contact book for storing, searching, editing, and deleting contacts.

**Concepts**

* Tkinter GUI
* SQLite Database
* CRUD Operations
* SQL Search (LIKE)
* Listbox
* Event Handling

## Learning Objectives

After completing these projects, students will be able to:

* Build desktop applications using Tkinter.
* Design graphical user interfaces.
* Connect Python applications to SQLite databases.
* Perform Create, Read, Update, Delete (CRUD) operations.
* Manage application data using SQL.
* Develop real-world desktop applications from scratch.
