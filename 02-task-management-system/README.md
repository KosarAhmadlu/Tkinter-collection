# Task Manager

A beginner-friendly desktop application built with **Python**, **Tkinter**, and **SQLite** for managing daily tasks.

This project demonstrates how to build a graphical user interface connected to a local SQLite database while implementing basic CRUD operations.

---

## Features

* Add a new task
* View all tasks
* Mark tasks as completed
* Delete tasks
* Store tasks in an SQLite database
* Simple and user-friendly interface

---

## Technologies Used

* Python
* Tkinter
* SQLite3

---

## Concepts Covered

* Tkinter GUI
* Labels
* Entry Widgets
* Buttons
* Listbox
* Grid Layout
* SQLite Database
* CRUD Operations
* Functions

---

## Learning Objectives

After completing this project, students will be able to:

* Build desktop applications with Tkinter.
* Connect Python applications to SQLite databases.
* Perform Create, Read, Update and Delete (CRUD) operations.
* Manage application state using a database.
* Design user-friendly GUI applications.
