# Book Database Manager

A beginner-friendly desktop application built with **Python**, **Tkinter**, and **SQLite** for managing a collection of books.

This project demonstrates how to build a graphical user interface (GUI) connected to a local SQLite database while implementing basic CRUD operations.

---

## Features

* Add a new book
* View all books
* Search books
* Delete books
* Store data using SQLite
* Simple and clean Tkinter interface

---

## Technologies Used

* Python
* Tkinter
* SQLite3

---

## Concepts Covered

* Tkinter GUI
* Labels
* Entry Widgets
* Buttons
* Listbox
* Grid Layout
* SQLite Database
* CRUD Operations
* Functions

---

## Learning Objectives

After completing this project, students will be able to:

* Build desktop applications using Tkinter.
* Connect Python applications to SQLite databases.
* Perform Create, Read, Search and Delete operations.
* Organize GUI applications using functions.
* Design simple CRUD desktop applications.
