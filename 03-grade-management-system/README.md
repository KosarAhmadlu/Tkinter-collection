# Grade Management System

A beginner-friendly desktop application built with **Python**, **Tkinter**, and **SQLite** for managing student grades.

This project demonstrates how to create a graphical user interface (GUI) connected to a local SQLite database while performing CRUD operations and automatically evaluating student performance.

---

## Features

* Add student grades
* Display all records
* Delete selected records
* Automatically calculate grade status
* Store data in an SQLite database
* Display data using a Treeview table

---

## Technologies Used

* Python
* Tkinter
* ttk.Treeview
* SQLite3

## Learning Objectives

After completing this project, students will be able to:

* Build desktop GUI applications using Tkinter.
* Connect Python applications to SQLite databases.
* Display database records using ttk.Treeview.
* Perform Create, Read and Delete operations.
* Apply conditional logic to evaluate student performance.
