# Contact Management System

A beginner-friendly desktop application built with **Python**, **Tkinter**, and **SQLite** for managing personal contacts.

This project demonstrates how to create a GUI application connected to an SQLite database while implementing full CRUD (Create, Read, Update, Delete) functionality and search capabilities.

---

## Features

* Add new contacts
* View all contacts
* Search contacts by name
* Edit existing contacts
* Delete contacts
* Store data in an SQLite database
* Simple and easy-to-use interface

---

## Technologies Used

* Python
* Tkinter
* SQLite3

---

## Concepts Covered

* Tkinter GUI
* Labels

* Buttons
* Listbox
* Grid Layout
* SQLite Database
* CRUD Operations
* Search Functionality
* Functions

---

## Learning Objectives

After completing this project, students will be able to:

* Build desktop applications using Tkinter.
* Connect GUI applications to SQLite databases.
* Implement full CRUD operations.
* Search records using SQL queries.
* Manage data through a graphical user interface.
